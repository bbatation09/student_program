2. 성적 출력

4. 화면 나가기

6. 성적 삭제: 입력한 학생이름과 목록 속 학생이름과 비교해서 일치하는 학생을 찾아 해당 학생에 관련 정보를 삭제함
     
7. 성적 수정: 입력한 학생이름과 목록 속 학생이름과 비교해서 일치하는 학생을 찾아 해당 학생에 관련 정보를 삭제함
